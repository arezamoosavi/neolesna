# neolesna
twitter stream grafana monitoring

## Installation

for creating the project:
```bash
docker-compose up --build -d
```
to stop it
```bash
docker-compose down -v
```

## Usage

to see grafana panel:
```bash
http://0.0.0.0:3000
```
user: admin
password: password
